# tgs_utsegi
tgs uts
Nama Aplikasi = Pemesanan Pizza
Deskripsi Singkat = Aplikasi Pemesanan Pizza adalah aplikasi yang memungkinkan pengguna untuk memesan pizza secara cepat dan mudah. Pengguna dapat memilih jenis pizza, ukuran, topping tambahan, jumlah pesanan, serta melihat total harga secara otomatis. Aplikasi ini dirancang sederhana dengan tampilan yang mudah dipahami sehingga cocok digunakan untuk latihan pengembangan aplikasi.

⭐ Fitur-Fitur
1. Menu Pizza
Menampilkan daftar pizza (Pepperoni, Margherita, BBQ Chicken, dll.)
Informasi harga masing-masing pizza.

2. Pilihan Ukuran
Small
Medium
Large

3. Topping Tambahan
Extra Cheese
Sausage
Mushroom
Beef
Onion

4. Keranjang Pesanan
Menambahkan pizza ke dalam keranjang.
Menghapus item pesanan.
Mengedit jumlah pesanan.

5. Perhitungan Total Harga
Total harga otomatis berdasarkan:
jenis pizza
ukuran
topping tambahan
jumlah pesanan

6. Halaman Checkout
Menampilkan detail pesanan.
Form data pelanggan (nama, alamat, nomor telepon).
Tombol konfirmasi pesanan.

7. Riwayat Pesanan (opsional)
Menyimpan pesanan sebelumnya.

▶️ Cara Menjalankan Aplikasi (Flutter / Mobile Version)

Jika aplikasi dibuat dengan Flutter:

1. Install Flutter
Download & install Flutter:
https://flutter.dev

Cek versi:
flutter --version

2. Clone atau Download Project
git clone https://github.com/username/pemesanan-pizza.git
cd pemesanan-pizza

4. Install Dependencies
flutter pub get

5. Jalankan Aplikasi
Pada emulator atau perangkat Android:
flutter run
